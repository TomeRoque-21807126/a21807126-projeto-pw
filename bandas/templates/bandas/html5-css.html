Seletores CSS

Os seletores CSS são utilizados para selecionar os elementos HTML que você deseja estilizar. Aqui estão alguns dos principais tipos de seletores CSS:

Seletor Utilizado Comentário
Elemento (tag) check_circle Seleciona todos os elementos de um tipo específico, por exemplo, p para parágrafos. Utilizado em Página 1.
ID check_circle Seleciona um elemento único com um ID específico. Utilizado em Home.
Classe check_circle Seleciona todos os elementos com uma classe específica. Utilizado em Página 3.
Atributo check_circle Seleciona elementos baseados em um atributo ou valor de atributo. Utilizado em Página 4.
Pseudo-classe check_circle Seleciona elementos em um estado específico, como :hover. Utilizado em Página 5.
Pseudo-elemento check_circle Seleciona uma parte específica de um elemento, como ::before. Utilizado em Página 6.

Combinadores de seletores

Combinador Utilizado Comentário
Descendente (espaço) check_circle Seleciona elementos que são descendentes de outro elemento. Utilizado em Página 7.
Filho (>) check_circle Seleciona elementos que são filhos diretos de outro elemento. Utilizado em Página 8.
Adjacente (+) check_circle Seleciona elementos que são imediatamente adjacentes a outro elemento. Utilizado em Página 9.
Irmão geral (~) check_circle Seleciona todos os elementos que são irmãos de um elemento anterior. Utilizado em Página 10.
